# PerfectLayout AI

PerfectLayout AI é um aplicativo de otimização de layout que usa heurísticas inteligentes para posicionar móveis dentro de um ambiente, buscando o melhor aproveitamento do espaço.

## Como executar (forma única e recomendada)

1. Garanta que o **Python 3** esteja instalado no Windows.
2. Execute o arquivo **`run_gui.bat`**.

> Esta é a forma suportada e ideal de iniciar o aplicativo.

## O que o aplicativo faz

- Analisa dimensões do ambiente.
- Interpreta itens e quantidades.
- Utiliza um mecanismo de otimização inspirado em inteligência artificial para gerar um layout com menor sobreposição e melhor aproveitamento do espaço.

## Dicas rápidas

- Informe medidas reais para maior precisão.
- Ajuste a lista de itens para explorar variações de layout.
